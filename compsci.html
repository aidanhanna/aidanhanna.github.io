Computer Science

Through coursework and independent study I have learned a wide variety of languages, from high-level, dynamically-typed, and interpreted python to lower-level, statically-typed, and compiled C++. Below is a compilation of every language I have learned, how I learned it, and when I’ve used it.

C++

Online C++ tutorials were my first introduction to programming. These taught me the basics of the language, and since it was my first programming language, they taught me the basics of programming as a whole as well. My most recent C++ experience is a course on data structures which covered structures as simple as arrays and vectors to B-plus trees and hash maps. This course reinforced the fundamentals of the language while teaching me new skills, and the enforcement of stricter coding standards prepared me to work more effectively in groups which has proved helpful in projects I’ve worked on with friends or needed help debugging. Due to my familiarity with the language, C++ is my preferred language for most projects. A small sample of my work is available here

C

My primary experience with C is its application in control systems for microcontrollers, specifically the Silicon Labs C8051F020 microcontroller. I used C for projects as simple as configuring inputs and outputs for a switch to control an LED and projects as complicated as creating a multi-state steering system using PID control. This gave me a solid grasp of control systems as well as C as a whole.

Python

My Computer Science I class taught python from the fundamentals to more advanced topics. This was my first formal computer science class and as such began my education on subjects like readability and efficiency, very important concepts that had previously been neglected. Python is my preferred language for quick prototyping or repeated calculations due to the ease and brevity of its syntax. Because we used the pillow module in class, python is also the language I turn to for creating things like the banner image at the top of each page was created using various features of the pillow module.
